# Where agent instructions live

Project guidance for AI assistants is centralized here:

- **`AGENTS.md`** (repo root) — navigation map for generic coding agents (e.g. Cursor).
- **`CLAUDE.md`** (repo root) — navigation map for Claude Code.

Detailed reference shards live under **`docs/reference/`** (see [Documentation hub](../README.md)).

There is **no** separate `.cursorrules` or `.github/copilot-instructions.md`; follow **`AGENTS.md` / `CLAUDE.md`** and the linked docs unless a task explicitly overrides them.
